# Python More Data Structures

This project involves learning and using Python's more advanced data structures including:
- Sets
- Dictionaries
- Lambda functions
- Map, Filter, and Reduce

## Task 0 - Squared Simple
- Write a function `square_matrix_simple` that computes the square value of all integers in a 2D matrix.

## Requirements
- All files must be executable.
- Code should adhere to **Pycodestyle** guidelines.
- Files should end with a newline.
