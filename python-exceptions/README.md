# Python Exceptions Handling

This project demonstrates how to handle exceptions in Python. It focuses on understanding the difference between errors and exceptions, as well as how to properly catch and handle exceptions in Python.

## Task 0 - Safe List Printing
- This function prints a specific number of elements from a list, handling exceptions such as `IndexError`.

## Requirements
- All files should follow the pycodestyle style guide.
- The files must be executable.
- Code must handle exceptions appropriately.
