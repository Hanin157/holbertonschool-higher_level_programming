# Print List Integer

This project defines a function `print_list_integer(my_list=[])` that prints all integers of a list, one integer per line.

The function uses the `str.format()` method for printing the integers in the specified format.

## Usage

- The list provided to the function should only contain integers.
- The function prints each integer of the list on a separate line.

Example:

```python
my_list = [1, 2, 3, 4, 5]
print_list_integer(my_list)
